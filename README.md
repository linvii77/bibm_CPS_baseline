# BIBM CPS Baseline Rerun Package

This repository records the rerun package for:

- Synapse CPS baseline, 20% labels
- AMOS MagicNet baseline, 10 labeled cases

The repository contains reproducibility metadata, data split files, evaluation
scripts, and final DSC/ASD/HD95/NSD result tables. Large `.pth` checkpoint
tensors are documented in `docs/WEIGHTS_MANIFEST.md` and should be stored with
Git LFS or GitHub release assets if they need to be published.

- `docs/REPRODUCE_FROM_ZERO.md`
- `docs/WEIGHTS_MANIFEST.md`
- `results/final_rerun_summary_dsc_asd_hd95_nsd.md`
- `results/final_rerun_detailed_dsc_asd_hd95_nsd.md`
